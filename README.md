## Featured Projects

### [TAILORAI](https://github.com/mrcoffeebean14/tailorai) — Multi-Agent Resume Tailoring System &nbsp;·&nbsp; [🔗 Live Demo](https://tailorai-xi.vercel.app/)
> LangGraph · FastAPI · Next.js · Docker · OpenAI / Claude / Gemini / Groq / Ollama

A multi-agent pipeline that analyzes job descriptions and rewrites resumes to match them. Built a provider-agnostic LLM layer so the same workflow runs on any model. Cuts tailoring time by ~70–80%. Full stack: FastAPI backend, Next.js frontend, Docker-composed, with support for both cloud APIs and local Ollama models.

---

### [LumenRTC](https://github.com/mrcoffeebean14/lumenrtc) — Peer-to-Peer Video Calling &nbsp;·&nbsp; [🔗 Live Demo](https://video-call-stremo-lumenrtc-1.onrender.com/)
> WebRTC · Node.js · Socket.io

Browser-to-browser video calling without a media relay server. Built the signaling layer with Socket.io and handled NAT traversal using STUN. No external media servers.

---

### [Explora](https://github.com/mrcoffeebean14/explora) — Places Explorer &nbsp;·&nbsp; [🔗 Live Demo](https://majorproject-llja.onrender.com/listings)
> Node.js · Express · MongoDB · GeoJSON · EJS

Location discovery app with geospatial queries via MongoDB GeoJSON, Mongoose middleware for data integrity, and session-based auth. Full MEN stack.

---

### [AI News Summarizer](https://github.com/mrcoffeebean14/ai-news-summarizer) — RAG Pipeline
> FastAPI · Docker · AWS Polly · RAG

Retrieval-augmented pipeline that ingests, chunks, and summarizes news articles with audio output via AWS Polly. Validated across 500+ articles. Built for accuracy and latency under realistic load.

### [Caption-AI](https://github.com/mrcoffeebean14/caption-ai) — Image Captioning Model
> ResNet50 · LSTM · Python

CNN+RNN image captioning pipeline. ResNet50 encodes the image; an LSTM decoder generates the caption. Trained end-to-end on a standard captioning benchmark.
